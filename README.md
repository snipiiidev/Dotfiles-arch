## Installation
Clone the repository

```
git clone https://github.com/snipiiidev/Waybar-Arch-.git
```
```
cd Waybar-Arch
```
Copy the configuration files to your system:
```
cp -r Hyprland ~/.config/
cp -r waybar ~/.config/
cp -r wofi ~/.config/
cp -r r Terminal ~/.config /  
```
This is a custom configuration; you may need to adapt it to your system. Some paths or settings may vary depending on your distribution.
